⚡ XP & Stat Scaling

Both XP and stats use exponential curves — easy at the start, increasingly hard as you level up. Adjust the base values and growth multipliers below. Changes apply immediately.

XP TO LEVEL UP

Formula: XP = Base × Scale^(Level−1)
At Scale 1.25: Lvl1→2 is 200 XP, Lvl10→11 is ~1490 XP, Lvl20→21 is ~17K XP.

STAT POINTS PER TIER

Formula: Pts = Base × Scale^Tier
Tier 1 is full gain (DR 100%), each tier reduces effectiveness by ~30%.
